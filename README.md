#  Weather Dashboard

A modern and responsive Weather Dashboard built using **HTML, CSS, and JavaScript**.  
The application uses the **OpenWeatherMap API** to display real-time weather information with dynamic weather-based themes.

##  Live Features

- Search weather for any city worldwide
-  Real-time temperature information
-  Current weather conditions
-  5-Day weather forecast
- Humidity details
-  Wind speed
- Atmospheric pressure
- Visibility
-  Sunrise and sunset time
-  Dynamic background changes based on weather
-  Loading animation
-  Fully responsive design



##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
- Font Awesome Icons
- Google Fonts
